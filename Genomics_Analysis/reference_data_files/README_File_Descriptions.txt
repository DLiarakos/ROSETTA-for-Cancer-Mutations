ALL_CASES_LIST_PID: lists all cases counted by the author DL.
All_counts_DM.txt: is the genomic matrix calculated by the author DL.
MutationsPipelines_CompareWithDavid.ipynb: is a jupyter notebook that inputs the above files and compares with equivalent output from first author GM's code.
Homo_sapiens_gene_info_GM.txt: is a list of gene names. The list is used to find gene aliases and combine them and standardize gene names in GM's implementation code shared in parent folder. 
Table2_v6.xlsx: contains an upgraded supplementary data Table 2 from the main manuscript. This table is used to remove redundant cases across several studies by retaining data from the temporally latest study and dropping all previous copies of the same patient data.
